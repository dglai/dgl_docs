dgl.distributed.load_partition_book

dgl.distributed.load_partition_book(part_config, part_id)[source]

Load a graph partition book from the partition config file.

Parameters:

  • part_config (str) – The path of the partition config file.

  • part_id (int) – The partition ID.

Returns:

  • GraphPartitionBook – The global partition information.

  • str – The graph name

  • dict – The node types

  • dict – The edge types
